# Multi-Step Registration Form (React + TypeScript + Tailwind)

## Overview

This project is a **Multi-Step Registration Form** built with **React (TypeScript)**.  
It demonstrates **complex form state management** using **React Context**, so that user input is **preserved** when navigating back and forth between steps.

Features:

- Step 1: Profile (Name & Email)
- Step 2: Experience (Years & Skills)
- Step 3: Review & Submit
- Form state managed globally via **React Context**
- Preserves input values when moving between steps
- Final "Submit" button logs the entire form object to the console
- Modern UI using **Tailwind CSS**
- Fully responsive and centered layout
